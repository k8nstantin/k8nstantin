## AI & Autonomous Agent Development

Created **[OpenLoom](https://github.com/k8nstantin/openloom)** — a spec-driven development framework where autonomous AI agents build software from structured specifications. Products break into manifests (specs), manifests break into tasks with dependency chains. The framework spawns independent agent sessions, feeds them the spec and context from every previous session, and a server-side watcher audits every completed task — commits, build, deliverables. The agent can't see it and can't bypass it. Every conversation recorded, every tool call captured, every dollar and line of code tracked with real-time productivity scoring. 19,000+ lines of Go, 40+ MCP tools, 70+ API endpoints, 16-tab dashboard, peer-to-peer sync. Single binary.

Built **Schema Intelligence** at Gryphon AI — an AI-powered knowledge network over 17,000+ MySQL database objects across 27 databases and 5 servers. Gemini 2.5 Pro summarizes every table, procedure, view, event, and trigger. Vertex AI Vector Search enables semantic queries. RAG chatbot with three modes: explore, impact analysis, migration planning. Multi-turn chat, source citations, session persistence. Full pipeline runs as a single serverless Dataproc job — $30 rebuild.

Designed a **fully serverless Apache Iceberg lakehouse** on GCP — Dataproc Serverless, BigLake REST Catalog, BigQuery federation. Zero persistent clusters, zero idle costs, modular Terraform.
