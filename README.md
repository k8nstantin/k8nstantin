## What I've Built

### OpenLoom — The AI Development Framework

**The first spec-driven development framework for autonomous coding agents.**

OpenLoom is a complete operating system for AI-powered software development. You define what you want built as structured specifications. Autonomous AI agents pick up the work, execute independently, and deliver — while a server-side auditor verifies every commit, every build, every deliverable. No human babysitting. Full cost transparency.

This isn't a chatbot wrapper or a prompt toolkit. This is a production-grade framework:

- **Spec-Driven Architecture** — Products, manifests, and tasks form a hierarchical build plan with dependency chains. Agents receive the full spec as context.
- **Autonomous Agent Execution** — Spawns independent Claude Code, Cursor, or any MCP-compatible agent. Each runs in its own branch, commits its own code, creates its own PR.
- **Independent Watcher Auditing** — Server-side auditor the agent cannot see, override, or bypass. Three gates: commits exist? Code compiles? Deliverables addressed? Fail any gate and the task is downgraded — the agent has no say.
- **Visceral Rules** — Non-negotiable operating constraints every agent must acknowledge. Violations flagged automatically as amnesia.
- **Real-Time Cost & Efficiency Analytics** — Every dollar spent, every turn taken, every line of code written — tracked per task, per agent, per day. Productivity scoring, first-attempt success rates, cost per completion, 7-day trends. You see exactly what autonomous development costs and produces.
- **Shared Memory** — Decisions, patterns, bugs, and constraints persist across all sessions and agents. No more repeating yourself.
- **Peer-to-Peer Sync** — Multiple machines discover each other via mDNS and sync using Automerge CRDTs. Fully decentralized.
- **16-Tab Dashboard** — Real-time: tasks, costs, productivity, compliance, conversations, memories, manifests, products, DAG visualization, watcher audits.

**19,000+ lines of Go. 40+ MCP tools. 70+ REST API endpoints. 31 SQLite tables. Single binary.**
