### Featured Project

#### [OpenLoom](https://github.com/k8nstantin/openloom) — Spec-Driven Development Platform

A full autonomous development platform for coding agents. Write specs, and independent AI agents build them — with persistent memory, compliance enforcement, and independent execution auditing.

**What it does:**
- **Products > Manifests > Tasks** — hierarchical spec-driven development
- **Autonomous Execution** — spawns independent Claude Code / Cursor sessions from specs
- **Watcher** — server-side auditor that agents cannot override (git, build, manifest compliance gates)
- **Visceral Rules** — non-negotiable constraints every agent must follow
- **Shared Memory** — decisions, patterns, and context persist across all agent sessions
- **Peer Sync** — multiple machines discover each other and sync via Automerge CRDTs
- **Dashboard** — 16-tab UI with real-time task monitoring, cost analytics, productivity scoring

**Tech:** Go, SQLite, Cytoscape.js, MCP (Model Context Protocol), Automerge CRDT, mDNS
